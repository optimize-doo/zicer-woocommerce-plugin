# Changelog

All notable changes to ZICER WooCommerce Sync will be documented in this file.

## [1.0.0] - 2024-12-10

### Added
- Initial release
- Product synchronization with ZICER marketplace
- Category mapping between WooCommerce and ZICER
- Background queue processing for bulk operations
- Real-time sync on product updates
- Image synchronization support
- Price conversion settings
- Product condition mapping
- Multi-language support (English, Bosnian)
- Automatic plugin updates from GitHub
